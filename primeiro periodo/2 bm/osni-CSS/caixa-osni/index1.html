Caixa com bordas arredondadas e sombra externa

2

4

Caixa com borda dupla e sombra em camadas

Sobre este exercício:

Este exercício demonstra diferentes propriedades CSS aplicadas a quatro caixas:

  • Box Model: Cada caixa utiliza diferentes tipos de bordas, preenchimentos e margens
  • Border-radius: Diferentes técnicas de arredondamento de cantos são aplicadas
  • Box-shadow: Sombras externas e internas com diferentes configurações
  • Transformações: Cada caixa tem uma transformação diferente no evento hover
  • Responsividade: O layout se adapta a telas menores usando media queries

Experimente redimensionar a janela do navegador para ver o comportamento responsivo e passe o mouse sobre as caixas para ver os efeitos de hover.
